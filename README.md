# Project Name: Editorial Portfolio

A minimal, responsive portfolio website built while learning modern web development layout techniques and scroll animations.

## 🚀 Tech Stack

- **Framework:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion

## 🛠️ Features

- Infinite running logo marquee line.
- Smooth scroll fade-in animations using Framer Motion viewports.
- Responsive 12-column CSS Grid layout for desktop and mobile displays.

## ⚙️ How to Run Locally

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
